# MTT GTOサバイバル — 5〜30BB中盤戦トレーナー

ポーカートーナメント(MTT)の中盤戦に特化したGTO学習アプリ。
チップが尽きるまでフルハンド(リバー・ショーダウンまで)をプレイし、すべての判断がGTO基準で採点されます。

**プレイはこちら: https://tkchemicalwarehouse.github.io/poker-gto-trainer/**

## 起動方法

上のURLをブラウザで開くか、`index.html` をダブルクリック。インストール不要。成績はブラウザ(localStorage)に保存されます。
ホーム画面で **📱スマホ / 🖥タブレット・PC** のレイアウトを選択できます(初回は画面幅で自動判定)。

## ルール設定

| 項目 | 設定 |
|---|---|
| テーブル | 9人 |
| ブラインド | 2,000/4,000 スタート、**2周(18ハンド)ごとにアップ** |
| アンティ | BBアンティ = BB額 |
| スタック | 全員 5〜30BB のランダム(自分も) |
| ボット入れ替え | バストしたボットは新規5〜30BBボットと交代(常に9人) |
| ベットサイズ | ソルバー風抽象化: プリフロップ 2.2BB or オールイン / ポストフロップ 33%・66%・オールイン |
| 効果音 | WebAudio合成(🔊ボタンでミュート可) |

## モード

- **トーナメント**: バストするまでプレイ。バスト時に「ミス由来か、分散か」を分解表示
- **GTOシミュレーション**: GTOボットに自動プレイさせ、生存ハンド数の分布をヒストグラム表示(「完璧に打っても飛ぶ」の体感用)
- **成績**: 挑戦回数・平均生存・GTO一致率・ブランダー数・分散バスト率など

## 採点

| 判定 | 意味 |
|---|---|
| ✓ GTO通り | 最高頻度のアクション |
| ✓ OK(混合戦略) | GTOが混ぜる選択肢のひとつ |
| △ 僅かなミス | EV損失小(レンジ境界近辺など) |
| ✗ ブランダー | 明確なEV損失。理由とレンジ・実数を表示 |

オールイン関連の判断は、相手レンジに対する**モンテカルロ・エクイティ vs 必要勝率**でEVを実数計算して採点します。

## GTOの中身(正確性について)

- **プリフロップ**: ナッシュ均衡チャート(オープンジャム/オープンレイズ/リジャム/BBディフェンス、HRC・GTO Wizard系チャートの近似)。`js/ranges.js` にレンジ文字列として格納されており、編集して精度を上げられます
- **オールインへのコール**: リアルタイムエクイティ計算による正確なEV判定
- **ポストフロップ**: 完全GTOはソルバー級計算が必要なため、ソルバー傾向(レンジベット・SPR別コミット基準・セミブラフ選択)を模したヒューリスティック+エクイティ計算による近似

## ファイル構成

```
index.html        画面
css/style.css     スタイル
js/engine.js      カード・役判定・モンテカルロエクイティ
js/ranges.js      GTOレンジデータ(編集可) + パーサー
js/strategy.js    戦略エンジン(ボットの脳 = コーチの採点基準)
js/poker.js       トーナメントエンジン(ベッティング・サイドポット・ショーダウン)
js/coach.js       採点・解説・レンジグリッド生成
js/app.js         UI制御・シミュレーション・成績
tests/run.cjs     ヘッドレステスト: node tests/run.cjs
```

## テスト

```
node tests/run.cjs
```

役判定・レンジパーサー・エクイティ精度(AA vs KK ≈ 82% 等)・戦略判断・ボット同士のフルトーナメント完走を検証します。
